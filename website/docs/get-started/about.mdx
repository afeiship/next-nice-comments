# 关于
> 一个可以方便写样式的工具。

### 安装
> 直接用 `npm` 或者 `yarn` 安装。

```bash npm2yarn
npm install @jswork/styled-box
```

### 配置

```js
import View from '@jswork/styled-box';
import styled from 'styled-components';

// 默认配置中添加插件
Object.assign(View.defaultProps, {
  styled,
  plugins: [
    require('@jswork/styled-plugin-absolute-center').default,
    // ...
  ],
});
```
